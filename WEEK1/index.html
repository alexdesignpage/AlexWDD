Week01 The Paradigm Shift

The rise of the mobile devises changes the way we share data. This changes the online traffic. Mobile First is the key with this amount of traffic. Responsive retrofitted websites are good but not enough. When you design for the mobile first you are building for the future.

Things to remember:

  • Desktop sales declining.
  • You see lot of people using their phones.
  • Start small scale up
  • Mobile is not a fad.

Mobile Users

We need to watch what users are doing to create a better design.

  • Be There
  • Be Useful
  • Be Quick

Reachability Matters

  • Speed Matters
  • Reduce Image usage
  • SVG formats images
  • Optimize and Minify CSS and JS files
